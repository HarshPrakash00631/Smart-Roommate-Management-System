# Smart-Roommate-Management-System
A full-stack roommate management system for expense tracking, balance calculation, and automated cleaning scheduling using Next.js and MongoDB.
## 🧠 Smart Roommate Management System

A full-stack web application designed to simplify shared living by managing expenses and automating cleaning schedules with fairness-driven logic.

### 🚀 Features

* 💸 **Expense Management** – Add, edit, and delete shared expenses with real-time balance calculation
* ⚖️ **Automatic Balance Engine** – Splits expenses and computes who owes whom instantly
* 🧹 **Smart Cleaning Scheduler** – Rotates cleaning duties using a deterministic date-based algorithm
* 👥 **Dynamic Roommate Management** – Add and remove roommates with instant UI updates
* ⚡ **Responsive UI** – Glassmorphism-based design with smooth transitions using Framer Motion

### 🏗️ Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS
* **Backend:** Next.js API Routes
* **Database:** MongoDB
* **Animations:** Framer Motion

### 🧩 Architecture Highlights

* RESTful API design with dynamic routing
* Optimized state management for real-time UI updates
* Modular component structure with reusable UI elements

### 📌 Future Improvements

* Settle-up feature for clearing balances
* Advanced expense splitting (custom/percentage)
* Cleaning history and analytics

### 💡 Motivation
